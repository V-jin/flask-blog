# Flask Blog Service (KNU OSP Lab)

오픈소스 프로그래밍 실습을 위한 WEB 서비스.

------------

## Tech Stack
- **Language:** Python
- **Framework:** Flask
- **VCS:** Git / GitHub


## 페이지 설명

1. 메인 ("/")
    * 접속 시 보이는 페이지 입니다. 

2. 홈 ("/home")
    * 홈페이지 입니다.

3. 코스 ("/course")
    * 현재 수강 수업명을 볼 수 있습니다.
        * 작성일(26.03.13) 기준, 테스트 기준이라 빈약합니다. 추후 다른 수업들도 추가해 볼 수 있는 기능을 만들겠습니다.
